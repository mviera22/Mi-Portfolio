# Mi Portfolio - Manuela Viera

¡Bienvenido a mi portfolio personal! Soy Manuela Viera, una programadora Jr en formación, especializada en desarrollo de software con Java, automatización de procesos con n8n, y desarrollo web con HTML, CSS y JavaScript.

## Sobre el Proyecto

Este portfolio es una página web estática que muestra mi experiencia profesional, habilidades técnicas, proyectos y formas de contacto. Está diseñado con un tema oscuro moderno y es completamente responsivo.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica y accesible.
- **CSS3**: Diseño moderno con variables CSS, gradientes y animaciones.
- **JavaScript**: Interactividad básica (modal de contacto, validación de formularios).
- **Git & GitHub**: Control de versiones y hosting.

## Características

- Diseño responsive para móviles y desktop.
- Tema oscuro elegante.
- Modal de contacto con formulario funcional (integrado con Formspree).
- Secciones: Sobre Mí, Experiencia, Habilidades, Proyectos, Contacto.
- Accesibilidad: Skip links, alt texts, navegación por teclado.
